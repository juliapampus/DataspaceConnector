# Features
{: .fs-9 }

Find here an overview of functionalities, used IDS libraries, and integrated IDS components.
{: .fs-6 .fw-300 }

---

In this section, you can find an overview of currently implemented features, which is continuously 
updated. The Dataspace Connector is a Java application following basic Spring Boot functionalities 
and architecture patterns. It supports the IDS Information Model in the latest version by using the
corresponding Java library and integrates the IDS Connector Framework for identity and message
handling. It provides a REST API for loading, updating, and deleting IDS persisted resources.
